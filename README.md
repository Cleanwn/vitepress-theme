## 主题项目构建

项目拉取

```bash
git clone https://github.com/Cleanwn/vitepress-theme.git
```

依赖安装

```bash
pnpm install
```

引用包构建

```bash
pnpm build
```

文档项目启动

```bash
cd docs
npm run dev
```

文档项目构建

```bash
npm run build
```
